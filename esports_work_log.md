## 주요 결정 사항 요약

| 결정 | 이유 |
|------|------|
| PandaScore → cito API 전환 | PandaScore는 선수별 지표 데이터 제공에 제약이 있음 |
| DB 방식 포기 → API 직접 호출 | 샘플 데이터 날짜 불일치, 팀명 오류 등 데이터 신뢰성 문제 |
| LCK 2026 데이터 제거 | 데이터 정확성 확보 불가, 2025 기준으로만 운용 |
| 실시간 기능 미구현 | 라이브 API 미흡, 종료/예정 경기만 처리 |
| Gemini 연동 | 선수별 KDA + 데미지 기여도 기반 경기 요약 자동 생성 |
