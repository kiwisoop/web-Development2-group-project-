You are helping build our Web Programming 2 team project.

Project: Sports Analysis & Summary Website

Goal:
Build a web service that provides sports match lists, match details, stats, event timelines, AI match summaries, and favorite team features.

Important:
Use a real-service-like architecture with React + Spring Boot.
Do not use Thymeleaf.
Spring Boot should work as a REST API server.
React should handle the user interface.
